
# Vibeathon project

**Round 1 — URL Shortener (30 minutes)**

The first round required building a functional URL shortener within 30 minutes.
The application converts long URLs into short ones.
The system stores the original URL and maps it to a unique short code, which redirects users to the destination when accessed.

Features:
- Generate short links from long URLs  
- Custom slug support

**Round 2 — Dashboard & Authentication (45 minutes)**

In the second round, the goal was to enhance the platform with user management and link tracking.
I added a login system and a dashboard that allows users to manage all their shortened links.

Features Added:  
- User login system  
- Personal dashboard  
- Link history tracking  
- Click analytics  
- Enable / disable links  
- Edit or delete links  
- Click limits with usage progress

**Round 3 — AI Customer Support Agent**

The final round required building an AI-powered chatbot that answers customer queries.

Features:

- AI-powered support chat
- Context-aware responses
- Suggested questions
